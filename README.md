# simpleguitatune
一个极简的吉他调音器
纯客户端实现：无需任何后端，不加载第三方 JavaScript 库，体积轻量，加载速度快。

5品调音辅助：提示了常用的“5品调音法”（如按住6弦5品对齐5弦空弦音110.00Hz）。点击底部的快捷卡片，系统会自动配置对应的弦和品位组合，计算出精确的目标频率。

精准频率测算：不依赖简单的 FFT 峰值（该方式在低频弦如E2时分辨率非常低），而是使用了更适合单乐器测音的自相关算法（Autocorrelation），可以提供稳定至小数点的实时频率。

实时波形可视化：采用 Web Audio API 获取时域数据并在 Canvas 上绘制柔和的青色示波波形，弹奏琴弦时会有直观的声音起伏反馈。

直觉交互反馈：调音指示条中心代表完美音准（此时显示绿色并提示“完美”），往左代表音偏低（黄色，提示“拧紧 ⬆️”），往右代表音偏高（红色，提示“拧松 ⬇️”）。
