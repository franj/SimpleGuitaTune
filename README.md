# 🎸 SimpleGuitaTune

一个运行在浏览器中的极简吉他调音工具，无需安装，打开即用。

---

## 🔗 在线使用

👉 [**点击此处直接运行调音器**](https://franj.github.io/SimpleGuitaTune/index.html)

---

## ⚙️ 功能特点

*   **纯客户端实现**：无需任何后端支持，不加载第三方 JavaScript 依赖库。代码轻量，加载与响应迅速。
*   **5品调音辅助**：内置常用的“5品调音法”提示（例如按住6弦5品对准5弦空弦音110.00Hz）。点击底部的快捷卡片，系统会自动配置对应的弦和品位组合，并计算出目标频率。
*   **基音频率测算**：不依赖简单的 FFT 峰值分析（该方式在分析 E2 等低频弦时分辨率较低），而是使用了更适合单音测量的**自相关算法（Autocorrelation）**，提供更稳定的实时频率参考。
*   **实时波形可视化**：基于 Web Audio API 实时获取时域音频数据，并在 Canvas 上绘制波形图，为您的弹奏提供直观的视觉反馈。
*   **直觉交互反馈**：调音指示条中心代表基准音准（接近时显示绿色并提示“保持住”），偏左代表音偏低（黄色，提示“拧紧 ⬆️”），偏右代表音偏高（红色，提示“拧松 ⬇️”）。
